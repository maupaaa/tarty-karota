# Tarty karota

**Tarty Karota** to aplikacja która pomaga w eksploracji świata ezoterycznego, dzięki połączeniu 8bitowego interfejsu z ogromem wiedzy ezoterycznej. Projekt pozwala użytkownikom na zgłębianie symboliki kart, oferując zarówno interpretacje wybranych kart, jak i losowo generowane rozkłady odpowiadające na konkretny problem.

## Funkcje
    
-   **Dwa tryby:**
    
    -   **Losowy:** Pozwól algorytmowi wybrać karty za Ciebie.
        
    -   **Manualny wybór:** Samodzielnie wybierz karty, które chcesz poddać interpretacji (np. z fizycznego rozkładu).
        
-   **Różnorodne rozkłady:** Obsługa popularnych układów, takich jak:
    
    -   Karta dnia (1 karta)
        
    -   Przeszłość / Teraźniejszość / Przyszłość (3 karty)
        
    -   Krzyż celtycki a także inne, równie istotne, rozkłady
        
-   **Baza wiedzy:** Szczegółowe opisy Wielkich i Małych Arkanów, znaczenia poszczególnych kolorów, ciekawostki związane z historią tarota.
